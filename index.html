Introducing seeWat
§ The estimated number of Visually
Impaired people in the world is
285,000,000
§ Of those approximately 39 million
are completely blind.
§ seeWat is designed to help the
visually impaired read text in real
life settings.
Made by BraillePlusPlus
How does it
work?
The onboard Ultrasonic sensor awaits a hand wave or
other detection to begin running the program
The front facing, mounted camera takes a picture of
environment and saves it in the Raspberry Pi
Google Cloud Vision API is used to extract text from
image onboard the Raspberry Pi
The text is then converted to a MP3 file using another
Google API
The MP3 file is played to the Bluetooth speaker or
earbuds built in, with the program ready to run again.

Hardware
§ The entire hardware system is incorporated in the top
of a hard hat. This allows for a near universal fit for
head sizes and easy hardware installation.
§ Holes are cut out from the sides to allow for camera
mounting and ultrasonic sensor.
§ The use of the ultrasonic sensor was initially meant
for the LEAP gesture control however due to
resources we were limited to the ultrasonic sensor.
§ The Raspberry Pi 3B+ allowed us to incorporate the
libraries needed while also being the right size to fit in
the hardhat
